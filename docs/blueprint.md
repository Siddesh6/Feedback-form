# **App Name**: Feedback Insights Pro

## Core Features:

- Form Builder: Drag-and-drop interface for creating feedback forms with various question types (text, multiple choice, rating scales).
- Anonymous Submission: Allows users to submit feedback anonymously, ensuring their identity is protected. Admin/organizer cannot identify the respondent.
- Admin Dashboard: Provides an overview of total forms created, responses received, and active forms. Includes charts and graphs for data visualization.
- Real-time Analytics: Aggregates and displays feedback data in real-time, showing average ratings, trends, and response distributions.
- AI Sentiment Analysis: Analyzes text-based feedback using AI to determine the sentiment (positive, negative, neutral) expressed in the responses and act as a tool to find potentially critical feedback.
- Report Export: Allows admins to export feedback data and analytics reports in PDF or Excel format for further analysis and sharing.
- QR Code Generation: Generates QR codes for each feedback form, making it easy for users to access and submit feedback via mobile devices.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, authority, and professionalism.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body font: 'Inter' sans-serif for clean readability.
- Headline font: 'Space Grotesk' sans-serif for a modern feel.
- Use simple, modern icons from Material Design to represent different form elements and actions.
- Card-based layout to organize information clearly. Prioritize responsiveness and ensure accessibility on all devices.
